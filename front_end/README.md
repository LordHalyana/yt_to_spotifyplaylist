# YouTube to Spotify Sync Wizard

This project is a Vite React TypeScript app using Material-UI. It features a three-step wizard for Spotify/YouTube sync configuration and a live sync log using SSE.

## Features

- **Step 1:** Spotify credentials input (client ID, secret)
- **Step 2:** Sync config form (YouTube playlist URL, Spotify playlist ID, etc.)
- **Step 3:** Live sync log display using SSE connected to `GET /sync/{jobId}`

## Getting Started

1. Install dependencies:
   ```sh
   npm install
   ```
2. Start the development server:
   ```sh
   npm run dev
   ```

## Customization

- Update the backend endpoint for SSE in `SyncLogSSE.tsx` if needed.
- Add your own logic for job creation and sync as required.
